# Distributed-Cache-Simulator
CacheSim — Distributed cache simulator built with Java Spring Boot + React. Implements consistent hashing via a TreeMap ring with 150 virtual nodes, LRU eviction using LinkedHashMap, and thread-safe ops with ReentrantReadWriteLock. Features real-time dashboard, node management, MongoDB logging, and hit/miss analytics.

I have used Java Spring Boot for backend in this project.
